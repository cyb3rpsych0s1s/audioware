1/*!
2Playable chunks of audio that are loaded into memory all at once.
3
4To play a static sound, pass a [`StaticSoundData`] to
5[`AudioManager::play`](crate::AudioManager::play).
6
7```no_run
8use kira::{
9	AudioManager, AudioManagerSettings, DefaultBackend,
10	sound::static_sound::{StaticSoundData, StaticSoundSettings},
11};
12
13let mut manager = AudioManager::<DefaultBackend>::new(AudioManagerSettings::default())?;
14let sound_data = StaticSoundData::from_file("sound.ogg")?;
15manager.play(sound_data)?;
16# Result::<(), Box<dyn std::error::Error>>::Ok(())
17```
18
19Compared to streaming sounds, static sounds have lower CPU usage and shorter delays
20when starting and seeking, but they use a lot more memory.
21*/
