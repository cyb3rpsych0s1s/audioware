Expand description

Construct data to be included as part of an error. The data must require no arguments to be created.

impl GenerateImplicitData for Option<Backtrace>

Available on crate features std only.

Only create a backtrace when an environment variable is set.

This looks first for the value of RUST_LIB_BACKTRACE then RUST_BACKTRACE. If the value is set to 1, backtraces will be enabled.

This value will be tested only once per program execution; changing the environment variable after it has been checked will have no effect.

§Interaction with the Provider API

If you enable the unstable-provider-api feature flag, a backtrace will not be captured if the original error is able to provide a Backtrace, even if the appropriate environment variables are set. This prevents capturing a redundant backtrace.
