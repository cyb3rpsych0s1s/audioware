Enum Value

Source 
pub enum Value<T> {
    Fixed(T),
    FromModulator {
        id: ModulatorId,
        mapping: Mapping<T>,
    },
    FromListenerDistance(Mapping<T>),
}
Expand description

A value that a parameter can be linked to.

Variants§

§

Fixed(T)

A fixed value.

§

FromModulator

The value of a modulator.

Fields

§id: ModulatorId

The modulator to link to.

§mapping: Mapping<T>

How the modulator’s value should be converted to the parameter’s value.

§

FromListenerDistance(Mapping<T>)

The distance of the current spatial track to a listener.
